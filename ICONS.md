# Ícones PWA

Para o PWA funcionar completamente, você precisa criar os ícones PNG:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

Você pode:
- Usar o arquivo `icon-placeholder.svg` como base
- Converter para PNG usando ferramentas online ou:
  - [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
  - [RealFaviconGenerator](https://realfavicongenerator.net/)

Depois de criar os ícones, atualize o `public/manifest.json` para referenciá-los.
